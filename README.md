# 🌶️ Python Flask Crash Course

Welcome to the **Python Flask Crash Course**! This single-class repository is a fast-paced crash course on building backend web servers and APIs using **Flask**, a lightweight WSGI web framework for Python.

## 📁 File Structure
*   **`documentation.md`**: Complete theoretical breakdown, decorator routing logic, request parsing, response handling, and final reference code blocks.
*   **`interview_questions.md`**: Conceptual, non-coding interview questions and answers (easy to intermediate).
*   **`app.py`**: Incomplete live-coding template with `TODO` markers.

---

## 🛠️ Setup Instructions

To get started, follow these commands in your terminal:

1.  **Navigate to the Directory:**
    ```bash
    cd python-flask-crash-course
    ```

2.  **Create a Virtual Environment:**
    ```bash
    python3 -m venv venv
    ```

3.  **Activate the Virtual Environment:**
    *   **Mac/Linux:**
        ```bash
        source venv/bin/activate
        ```
    *   **Windows:**
        ```cmd
        venv\Scripts\activate
        ```

4.  **Install Dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

5.  **Run the Live Coding Server:**
    ```bash
    python app.py
    ```
